Learn HTML

HTML is the standard markup language for Web pages.

With HTML you can create your own Website.

HTML is easy to learn - You will enjoy it!

Easy Learning with HTML "Try it Yourself"

With our "Try it Yourself" editor, you can edit the HTML code and view the result:

Example:

<!DOCTYPE html>
<html>
<head>
  <title>My First HTML Page</title>
</head>
<body>

  <h1>This is a Heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>

Example Explained

The <!DOCTYPE html> declaration defines that this document is an HTML5 document.

The <html> element is the root element of an HTML page.

The <head> element contains meta information about the HTML page.

The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).

The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

The <h1> element defines a large heading.

The <p> element defines a paragraph.

What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag:

<tagname> Content goes here... </tagname>

The HTML element is everything from the start tag to the end tag:

<h1>My First Heading</h1>
<p>My first paragraph.</p>
Start tag Element content End tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
<br> none none
Note: Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an end tag!

Web Browsers

The purpose of a web browser (Chrome, Edge, Firefox, Safari) is to read HTML documents and display them correctly.

A browser does not display the HTML tags, but uses them to determine how to display the document:

View in Browser

HTML Page Structure

Below is a visualization of an HTML page structure:

<html>
  <head>
    <title>Page title</title>
  </head>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>
  </body>
</html>
Note: The content inside the <body> section will be displayed in a browser. The content inside the <title> element will be shown in the browser's title bar or in the page's tab.

HTML Text Formatting

HTML contains several elements for defining text with a special meaning.

Example

This text is bold

This text is italic

This is subscript and superscript

HTML Formatting Elements

Formatting elements were designed to display special types of text:

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

HTML <b> and <strong> Elements

The HTML <b> element defines bold text, without any extra importance.

Example:

<b>This text is bold</b>

The HTML <strong> element defines text with strong importance. The content inside is typically displayed in bold.

Example:

<strong>This text is important!</strong>

HTML <i> and <em> Elements

The HTML <i> element defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

Tip: The <i> tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.

Example:

<i>This text is italic</i>

The HTML <em> element defines emphasized text. The content inside is typically displayed in italic.

Tip: A screen reader will pronounce the words in <em> with an emphasis, using verbal stress.

Example:

<em>This text is emphasized</em>

HTML <small> Element

The HTML <small> element defines smaller text:

Example:

<small>This is some smaller text.</small>

HTML <mark> Element

The HTML <mark> element defines text that should be marked or highlighted:

Example:

<p>Do not forget to buy <mark>milk</mark> today.</p>

HTML <del> Element

The HTML <del> element defines text that has been deleted from a document. Browsers will usually strike a line through deleted text:

Example:

<p>My favorite color is <del>blue</del> red.</p>

HTML <ins> Element

The HTML <ins> element defines a text that has been inserted into a document. Browsers will usually underline inserted text:

Example:

<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>

HTML <sub> Element

The HTML <sub> element defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O:

Example:

<p>This is <sub>subscripted</sub> text.</p>

HTML <sup> Element

The HTML <sup> element defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font. Superscript text can be used for footnotes, like WWW[1]:

Example:

<p>This is <sup>superscripted</sup> text.</p>

HTML Tags

17 Mar 2025 | 15 min read

HTML tags are like keywords that define how a web browser will format and display the content. With the help of tags, a web browser can distinguish between HTML content and plain text.

HTML tags contain three main parts: opening tag, content, and closing tag. However, some tags are self-closing (unclosed tags).

When a browser reads an HTML document, it parses it from top to bottom and left to right. HTML tags are used to structure web pages and define their content's behavior and appearance.

Essential HTML tags are required so the browser can differentiate between regular text and HTML. You can use as many tags as needed based on your code’s requirements.

  • All HTML tags are enclosed within < > brackets.
  • Each tag performs a specific task.
  • Most tags require a closing tag </tag>, except self-closing tags.

Syntax:

<tag> content </tag>

HTML Tag Examples

<p> Paragraph Tag </p>
<h2> Heading Tag </h2>
<b> Bold Tag </b>
<i> Italic Tag </i>
<u> Underline Tag </u>

Unclosed HTML Tags

Some HTML tags do not have a closing tag. Common examples include:

  • <br> – Inserts a line break.
  • <hr> – Inserts a horizontal line.

HTML Meta Tags

DOCTYPE, title, link, meta, style

HTML Text Tags

<p>, <h1> to <h6>, <strong>, <em>, <abbr>, <blockquote>, <code>, <br> and more.

HTML Link Tags

<a>, <base>

HTML Image and Object Tags

<img>, <area>, <map>, <param>, <object>

HTML List Tags

<ul>, <ol>, <li>, <dl>, <dt>, <dd>

HTML Table Tags

<table>, <tr>, <td>, <th>, <thead>, <tbody>, <caption>

HTML Form Tags

<form>, <input>, <textarea>, <select>, <button>, <label>

HTML Scripting Tags

<script>, <noscript>

Note: We will see practical examples of these tags in upcoming chapters.

HTML Elements

An HTML element is defined by a start tag, some content, and an end tag.

The HTML element includes everything from the start tag to the end tag:

<tagname>Content goes here...</tagname>

Examples of HTML Elements

<h1>My First Heading</h1>
<p>My first paragraph.</p>
Start tag Element content End tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
<br> None None

Note: Some HTML elements have no content (like the <br> element). These are called empty elements and do not have a closing tag.

Nested HTML Elements

HTML elements can be nested, which means they can contain other elements. Every HTML document consists of nested HTML elements.

The following example contains four nested elements: <html>, <body>, <h1>, and <p>.

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Example Explained

  • <html> is the root element that wraps the whole document.
  • <body> contains the visible page content.
  • <h1> is a heading element.
  • <p> is a paragraph element.

Never Skip the End Tag

Some elements might display correctly without an end tag, but it's bad practice. Always use proper closing tags to avoid errors.

<html>
<body>

<p>This is a paragraph
<p>This is another paragraph

</body>
</html>

Warning: Never rely on missing end tags! It can cause unexpected behavior.

Empty HTML Elements

HTML elements with no content are called empty elements. For example:

<p>This is a <br> paragraph with a line break.</p>

HTML is Not Case Sensitive

HTML tags are not case-sensitive. <P> is the same as <p>.

However, it is recommended to use lowercase tag names, especially for compatibility with XHTML and modern coding standards.

At W3Schools and in best practice tutorials, lowercase is always preferred.

HTML Attributes

Attributes provide extra information about elements, like `href`, `src`, `alt`, and `style`.

HTML Attributes

HTML attributes provide additional information about HTML elements.

  • All HTML elements can have attributes.
  • Attributes are always specified in the start tag.
  • They usually come in name/value pairs like: name="value".

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the link’s destination:

<a href="https://www.w3schools.com">Visit W3Schools</a>

The src Attribute

The <img> tag is used to embed an image. The src attribute holds the path to that image:

<img src="img_girl.jpg">

Two ways to use src:

  • Absolute URL: src="https://www.example.com/image.jpg"
  • Relative URL: src="images/pic.jpg" or src="/images/pic.jpg"

Tip: Use relative URLs to keep your site flexible and domain-independent.

The width and height Attributes

Specify image size in pixels using width and height:

<img src="img_girl.jpg" width="500" height="600">

The alt Attribute

This provides alternate text when an image cannot be shown (due to errors or accessibility):

<img src="img_girl.jpg" alt="Girl with a jacket">

Example with missing image:

<img src="img_typo.jpg" alt="Girl with a jacket">

The style Attribute

Adds inline styling such as colors and fonts:

<p style="color:red;">This is a red paragraph.</p>

The lang Attribute

Used in the <html> tag to define the language of the page:

<html lang="en">
<html lang="en-US">

Use language + country codes for more accuracy (e.g., en-US).

The title Attribute

Shows a tooltip when you hover over the element:

<p title="I'm a tooltip">This is a paragraph.</p>

We Suggest: Use Lowercase Attributes

HTML is not case-sensitive, but W3C recommends lowercase for better consistency and XHTML compatibility:

Good:

<a href="https://www.w3schools.com/html/">Visit our HTML tutorial</a>

Bad:

<a href=https://www.w3schools.com/html/>Visit our HTML tutorial</a>

We Suggest: Always Quote Attribute Values

Although quotes are optional, always use them—especially when the value contains spaces.

Bad Example:

<p title=Description of W3Schools>

This will not display the tooltip correctly. Always use quotes!

HTML Anchor Tag

12 Mar 2025 | 2 min read

The HTML <a> (anchor) tag defines a hyperlink that links one page to another page. It can also link to files, locations on the same page, or external URLs.

The href attribute is the most important part of the anchor tag—it specifies the destination of the link.

Syntax

<a href="destination_url">Link Text</a>

Basic Example

<a href="second.html">Click for Second Page</a>

Opening Link in a New Tab using target Attribute

If you want to open the link in a new tab or window, use the target="_blank" attribute inside the anchor tag.

<!DOCTYPE html>
<html>
<head>
  <title>Link using target attribute</title>
</head>
<body>

  <p>Click on 
    <a href="https://www.tpointtech.com/" target="_blank">this-link</a> 
    to go on home page of Tpoint Tech.
  </p>

</body>
</html>
Note:
  • The target attribute should be used along with the href attribute.
  • If you don’t use target="_blank", the link will open in the same tab.

Appearance of Anchor Tags

  • Unvisited Link: Underlined and blue.
  • Visited Link: Underlined and purple.
  • Active Link: Underlined and red while being clicked.

HTML Image

12 Mar 2025 | 5 min read

The HTML <img> tag is used to display images on a web page. It is an empty tag, meaning it does not have a closing tag—it only contains attributes.

Syntax

<img src="url">

Here, src defines the image source (location). The URL can be relative or absolute.

Example

<h2>HTML Image Example</h2>
<img src="good_morning.jpg" alt="Good Morning Friends">

Organizing Images

It’s best practice to place images inside an images/ folder to keep your project organized.

<img src="images/good_morning.jpg" alt="Good Morning Friends">

Loading from Another Directory

<img src="E:/images/animal.png" height="180" width="300" alt="animal image">
Note: If the src URL is incorrect, the image won’t be displayed. Always use correct paths!

Important Attributes of <img> Tag

  • src: Specifies the path to the image (required).
  • alt: Provides alternate text if the image cannot be displayed.
  • width and height: Set image size (in pixels or %). Better handled with CSS.

Example with alt, height, and width:

<img src="animal.jpg" height="180" width="300" alt="animal image">
SEO Tip: Always include meaningful alt text to improve accessibility and SEO.

Spacing Attributes (Legacy, not recommended in modern HTML)

  • hspace: Adds horizontal spacing.
  • vspace: Adds vertical spacing.

Example:

<img src="good-morning.jpg" alt="Good Morning Friends" vspace="30" hspace="20">

Border Attribute

You can add a border around the image using the border attribute (deprecated in HTML5 — use CSS instead).

<img src="good-morning.jpg" alt="Good Morning Friends" border="3">

Use Image as a Link

Wrap the <img> tag inside an <a> tag to create a clickable image:

<a href="https://www.tpointtech.com/what-is-robotics">
  <img src="robot.jpg" height="100" width="100">
</a>

HTML Table

12 Mar 2025 | 4 min read

The HTML <table> tag is used to display data in a tabular format (rows × columns). Each row is defined by the <tr> tag, headers by <th>, and data cells by <td>.

Basic Table Example

<table>
  <tr><th>First_Name</th><th>Last_Name</th><th>Marks</th></tr>
  <tr><td>Sonoo</td><td>Jaiswal</td><td>60</td></tr>
  <tr><td>James</td><td>William</td><td>80</td></tr>
  <tr><td>Swati</td><td>Sironi</td><td>82</td></tr>
</table>

HTML Table Tags Summary

  • <table>: Defines a table.
  • <tr>: Table row.
  • <th>: Table header cell.
  • <td>: Table data cell.
  • <caption>: Table caption/title.
  • <thead>, <tbody>, <tfoot>: Groups table sections.
  • <col>, <colgroup>: For column styling.

Adding Borders with CSS

<style>
  table, th, td {
    border: 1px solid black;
    border-collapse: collapse;
  }
</style>

Adding Padding

<style>
  th, td {
    padding: 10px;
  }
</style>

Setting Table Width

Specify width in px or % using CSS:

table {
  width: 100%;
}

Colspan Example

<table>
  <tr>
    <th>Name</th>
    <th colspan="2">Mobile No.</th>
  </tr>
  <tr>
    <td>Ajeet</td>
    <td>123456</td>
    <td>789012</td>
  </tr>
</table>

Rowspan Example

<table>
  <tr><th>Name</th><td>Ajeet</td></tr>
  <tr><th rowspan="2">Mobile</th><td>123456</td></tr>
  <tr><td>789012</td></tr>
</table>

Table with Caption

<table>
  <caption>Student Records</caption>
  <tr><th>First Name</th><th>Last Name</th><th>Marks</th></tr>
  <tr><td>Vimal</td><td>Jaiswal</td><td>70</td></tr>
</table>

Alternate Row Styling

Use :nth-child() selectors to style even and odd rows.

<style>
  table#striped tr:nth-child(even) {
    background-color: #eee;
  }
  table#striped tr:nth-child(odd) {
    background-color: #fff;
  }
  table#striped th {
    background-color: gray;
    color: white;
  }
</style>

Tip

Note: Use div and CSS for page layout instead of tables. Tables are best for data only.

HTML Lists

12 Mar 2025 | 2 min read

HTML lists are used to display items in a structured format. There are three types of lists in HTML:

  • Ordered List - Displays items in numbered format using <ol>
  • Unordered List - Displays items in bullet format using <ul>
  • Description List - Displays terms and descriptions using <dl>
Note: Lists can be nested (i.e., a list inside another list).

Ordered List

An ordered list is created using the <ol> tag. Each list item starts with <li>.

<ol>
  <li>Aries</li>
  <li>Bingo</li>
  <li>Leo</li>
  <li>Oracle</li>
</ol>

Unordered List

An unordered list is created using the <ul> tag and shows bullet points by default.

<ul>
  <li>Aries</li>
  <li>Bingo</li>
  <li>Leo</li>
  <li>Oracle</li>
</ul>

Description List

This list type is perfect for dictionaries, glossaries, or name-value data. Use <dl>, <dt> (term), and <dd> (description).

<dl>
  <dt>Aries</dt>
  <dd>- One of the 12 horoscope signs.</dd>
  <dt>Bingo</dt>
  <dd>- A fun evening snack.</dd>
  <dt>Leo</dt>
  <dd>- Another zodiac sign.</dd>
  <dt>Oracle</dt>
  <dd>- A multinational tech corporation.</dd>
</dl>

Nested List

Lists can be nested inside each other. Here's an example combining an ordered list with nested unordered lists:

<ol>
  <li>Delhi
    <ul>
      <li>New Delhi</li>
    </ul>
  </li>
  <li>Haryana
    <ul>
      <li>Chandigarh</li>
    </ul>
  </li>
  <li>Gujarat
    <ul>
      <li>Gandhinagar</li>
    </ul>
  </li>
</ol>
Tip: Use CSS to customize list styles using properties like list-style-type, padding, and margin.

HTML Forms

12 Mar 2025 | 4 min read

HTML forms are used to collect user input and send it to a server for processing. Forms are a crucial part of interactive websites.

The <form> Element

The <form> tag wraps all form elements:

<form>
  ...form elements...
</form>

The <input> Element

<input> is the most commonly used form element. Its type attribute controls its behavior.

Type Description
text Single-line text input field
radio Select one option
checkbox Select zero or more options
submit Submit form data

Text Fields

Used for inputting text:

<form>
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname"><br>
  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname">
</form>

The <label> Element

<label> improves accessibility and makes forms more user-friendly. Use the for attribute to link it with an input by its id.

Radio Buttons

Let the user pick only one option from a set:

<form>
  <input type="radio" id="html" name="fav_language" value="HTML">
  <label for="html">HTML</label><br>

  <input type="radio" id="css" name="fav_language" value="CSS">
  <label for="css">CSS</label><br>

  <input type="radio" id="js" name="fav_language" value="JavaScript">
  <label for="js">JavaScript</label>
</form>

Checkboxes

Used to allow selecting multiple options:

<form>
  <input type="checkbox" id="bike" name="vehicle1" value="Bike">
  <label for="bike">I have a bike</label><br>

  <input type="checkbox" id="car" name="vehicle2" value="Car">
  <label for="car">I have a car</label><br>

  <input type="checkbox" id="boat" name="vehicle3" value="Boat">
  <label for="boat">I have a boat</label>
</form>

Submit Button

Used to send the form data to the server:

<form action="/action_page.php">
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname" value="John"><br>

  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname" value="Doe"><br><br>

  <input type="submit" value="Submit">
</form>

The name Attribute

The name attribute is important — only inputs with this attribute will send data when submitted.

Warning: If name is missing in <input>, its value will not be sent to the server.

HTML Iframes

12 Mar 2025 | 4 min read

The <iframe> tag in HTML is used to embed another webpage within the current webpage. It creates a rectangular frame where another document can be displayed.

Basic Syntax

<iframe src="URL"></iframe>

The src attribute specifies the URL of the page to embed.

Set Width and Height

You can control the size using either attributes or CSS.

Example (Pixels)

<iframe src="https://www.tpointtech.com/" height="300" width="400"></iframe>

Example (Percentage)

<iframe src="https://www.tpointtech.com/" height="50%" width="70%"></iframe>

Example (CSS)

<iframe src="https://www.tpointtech.com/" style="height:300px; width:400px;"></iframe>

Removing the Border

Use CSS to remove or style the border:

No Border

<iframe src="https://www.tpointtech.com/" style="border:none;"></iframe>

Custom Border

<iframe src="https://www.tpointtech.com/" style="border:2px solid tomato;"></iframe>

Targeting an iframe with a Link

You can link to an iframe by matching the link's target attribute with the iframe's name attribute:

<iframe src="new.html" name="iframe_a" height="300" width="100%"></iframe>
<p><a href="https://www.tpointtech.com" target="iframe_a">JavaTpoint.com</a></p>

Embedding YouTube Videos

You can embed videos directly from YouTube:

<iframe width="550" height="315" 
  src="https://www.youtube.com/embed/JHq3pL4cdy4" 
  frameborder="0" 
  allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" 
  allowfullscreen style="padding:20px;">
</iframe>

Attributes of <iframe>

Attribute Description
srcSpecifies the URL to embed
heightHeight in px or % (default: 150px)
widthWidth in px or % (default: 300px)
nameNames the iframe (used with links)
allowfullscreenEnables fullscreen option
sandboxApplies extra restrictions (e.g., no forms, scripts)
allow-scriptsAllows JavaScript inside iframe
allow-same-originTreat content as same-origin
srcdocEmbed HTML content directly
Note: frameborder and scrolling are not supported in HTML5.

HTML <marquee> Tag

Last Updated: 17 Apr, 2025

The <marquee> tag is used to create a scrolling effect (text or images) on a webpage. Though it's deprecated in HTML5, it can still be useful in legacy projects or simple demos.

Note: The <marquee> tag is not supported in modern HTML5 standards. Use CSS animations for better practice.

Syntax

<marquee>
  Your scrolling content
</marquee>

Attributes of <marquee>

Attribute Values Description
bgcolorColorBackground color of the marquee
directionleft, right, up, downScrolling direction
loopnumberHow many times to repeat the scroll (default: infinite)
heightpx or %Height of the marquee
widthpx or %Width of the marquee
hspacepxHorizontal spacing around the marquee
vspacepxVertical spacing around the marquee

JavaScript Methods

  • start() – Starts scrolling
  • stop() – Stops scrolling

Event Handlers

  • onstart – Triggered when scrolling begins
  • onfinish – Triggered after the final scroll
  • onbounce – Triggered when content bounces (if behavior is 'alternate')

Examples

Example 1: Right to Left Scrolling

<marquee bgcolor="green" direction="left">
  GeeksforGeeks - A computer science portal for geeks
</marquee>

Example 2: Bottom to Top Scrolling

<marquee bgcolor="green" direction="up">
  GeeksforGeeks - A computer science portal for geeks
</marquee>

Example 3: Top to Bottom Scrolling

<marquee bgcolor="green" direction="down">
  GeeksforGeeks - A computer science portal for geeks
</marquee>
Tip: For modern, accessible animation, prefer @keyframes and CSS animation instead of using <marquee>.
