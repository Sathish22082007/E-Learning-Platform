Learn CSS

CSS is used to apply styles.

CSS stands for Cascading Style Sheets.

It works with HTML to make websites beautiful and user-friendly.

CSS Introduction

CSS is the language we use to style a Web page.

What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Why Use CSS?

CSS is used to define styles for your web pages, including the design, layout, and variations in display for different devices and screen sizes.

CSS Example

body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}

CSS Solved a Big Problem

HTML was NEVER intended to contain tags for formatting a web page!

HTML was created to describe the content of a web page, like:

<h1>This is a heading</h1>

<p>This is a paragraph.</p>

When tags like <font> and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers.

Development of large websites, where fonts and color information were added to every single page, became a long and expensive process.

To solve this problem, the World Wide Web Consortium (W3C) created CSS.

Note: CSS removed the style formatting from the HTML page!

If you don't know what HTML is, we suggest that you read our HTML Tutorial.

CSS Saves a Lot of Work!

The style definitions are normally saved in external .css files.

With an external stylesheet file, you can change the look of an entire website by changing just one file!

CSS Selectors

A CSS selector selects the HTML element(s) you want to style.

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

We can divide CSS selectors into five categories:

  • Simple selectors (select elements based on name, id, class)
  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class selectors (select elements based on a certain state)
  • Pseudo-elements selectors (select and style a part of an element)
  • Attribute selectors (select elements based on an attribute or attribute value)

This page will explain the most basic CSS selectors.

The CSS element Selector

The element selector selects HTML elements based on the element name.

Example:

p {
  text-align: center;
  color: red;
}

The CSS id Selector

The id selector uses the id attribute of an HTML element to select a specific element.

The id of an element is unique within a page, so the id selector is used to select one unique element!

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

Example:

#para1 {
  text-align: center;
  color: red;
}
Note: An id name cannot start with a number!

The CSS class Selector

The class selector selects HTML elements with a specific class attribute.

To select elements with a specific class, write a period (.) character, followed by the class name.

Example:

.center {
  text-align: center;
  color: red;
}

You can also specify that only specific HTML elements should be affected by a class.

Example:

p.center {
  text-align: center;
  color: red;
}

HTML elements can also refer to more than one class.

Example:

<p class="center large">This paragraph refers to two classes.</p>
Note: A class name cannot start with a number!

The CSS Universal Selector

The universal selector (*) selects all HTML elements on the page.

Example:

* {
  text-align: center;
  color: blue;
}

The CSS Grouping Selector

The grouping selector selects all the HTML elements with the same style definitions.

Look at the following CSS code (the h1, h2, and p elements have the same style definitions):

h1 {
  text-align: center;
  color: red;
}

h2 {
  text-align: center;
  color: red;
}

p {
  text-align: center;
  color: red;
}

It will be better to group the selectors, to minimize the code.

To group selectors, separate each selector with a comma.

Example:

h1, h2, p {
  text-align: center;
  color: red;
}

How To Add CSS

When a browser reads a style sheet, it will format the HTML document according to the information in the style sheet.

Three Ways to Insert CSS

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

With an external style sheet, you can change the look of an entire website by changing just one file!

Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the <head> section.

Example:

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="mystyle.css">
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

An external style sheet can be written in any text editor, and must be saved with a .css extension.

The external .css file should not contain any HTML tags.

"mystyle.css"

body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}
Note: Do not add a space between the property value and the unit (e.g., 20px).
Incorrect: margin-left: 20 px;
Correct: margin-left: 20px;

Internal CSS

An internal style sheet may be used if one single HTML page has a unique style.

The internal style is defined inside the <style> element, inside the <head> section.

Example:

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background-color: linen;
    }

    h1 {
      color: maroon;
      margin-left: 40px;
    }
  </style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Inline CSS

An inline style may be used to apply a unique style for a single element.

To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

Example:

<!DOCTYPE html>
<html>
<body>

<h1 style="color:blue;text-align:center;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

</body>
</html>
Tip: Inline styles mix content with presentation and lose many advantages of CSS. Use them sparingly.

Multiple Style Sheets

If some properties have been defined for the same selector in different style sheets, the value from the last read sheet will be used.

Assume external CSS:

h1 {
  color: navy;
}

Then internal CSS:

h1 {
  color: orange;
}

If internal CSS is after the external link:

<head>
  <link rel="stylesheet" href="mystyle.css">
  <style>
    h1 { color: orange; }
  </style>
</head>

Then h1 color = orange

If internal CSS is before the external link:

<head>
  <style>
    h1 { color: orange; }
  </style>
  <link rel="stylesheet" href="mystyle.css">
</head>

Then h1 color = navy

Cascading Order

What style will be used when multiple styles are applied to the same element?

Styles "cascade" by the following priority order (from highest to lowest):

  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the <head> section)
  3. Browser default

✅ Inline style always overrides external and internal styles.

CSS Background

26 Mar 2025 | 2 min read

The CSS background property is used to define background effects on elements. There are 5 background properties that affect HTML elements:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

1) CSS background-color

This property is used to specify the background color of an element.

Example:

<style>
h2, p {
  background-color: #b0d4de;
}
</style>

Output:

My first CSS page.

Hello Javatpoint. This is an example of CSS background-color.

2) CSS background-image

This property sets an image as a background of an element.

Example:

<style>
body {
  background-image: url("paper1.gif");
  margin-left: 100px;
}
</style>
Note: Choose background images that contrast well with the text for better readability.

3) CSS background-repeat

By default, background images repeat both horizontally and vertically. You can control this behavior with background-repeat.

Repeat Horizontally:

background-repeat: repeat-x;

Example:

<style>
body {
  background-image: url("gradient_bg.png");
  background-repeat: repeat-x;
}
</style>

Repeat Vertically:

background-repeat: repeat-y;

Example:

<style>
body {
  background-image: url("gradient_bg.png");
  background-repeat: repeat-y;
}
</style>

4) CSS background-attachment

This property specifies whether the background image is fixed or scrolls with the rest of the page.

background: white url('bbb.gif');
background-repeat: no-repeat;
background-attachment: fixed;

5) CSS background-position

This property defines the initial position of the background image. Default is top-left. Possible values:

  • center
  • top
  • bottom
  • left
  • right
background: white url('good-morning.jpg');
background-repeat: no-repeat;
background-attachment: fixed;
background-position: center;

CSS Borders

The CSS border property allows you to define a border around HTML elements. You can specify the border’s width, style, and color.

Border Properties

CSS provides several border-related properties:

  • border-style
  • border-width
  • border-color
  • border (shorthand)

1) border-style

This property sets the style of the border. Possible values include:

  • none
  • solid
  • dotted
  • dashed
  • double
  • groove
  • ridge
  • inset
  • outset

Example:

p {
  border-style: solid;
}

2) border-width

Defines the thickness of the border. It can be specified in units like px, em, or using keywords like thin, medium, thick.

Example:

p {
  border-style: solid;
  border-width: 2px;
}

3) border-color

Sets the color of the border. You can use named colors, hex, RGB, or HSL values.

Example:

p {
  border-style: solid;
  border-color: red;
}

4) border (Shorthand)

The border shorthand property lets you define all three border properties in one line.

Syntax: border: <width> <style> <color>;

Example:

p {
  border: 2px dashed green;
}

Individual Borders

You can style each side of the border individually using:

  • border-top
  • border-right
  • border-bottom
  • border-left

Example:

p {
  border-top: 2px solid blue;
  border-right: 2px dashed red;
  border-bottom: 2px dotted green;
  border-left: 2px double orange;
}

Rounded Borders

Use border-radius to create rounded corners.

Example:

p {
  border: 2px solid black;
  border-radius: 10px;
}

CSS Border Radius

The border-radius property in CSS is used to create rounded corners for elements. You can apply it to all corners at once, or target each corner individually.

Basic Syntax

element {
  border-radius: value;
}

Example: All Corners Rounded

div {
  border: 2px solid #333;
  border-radius: 15px;
}
Rounded corners with border-radius: 15px

Different Radius for Each Corner

You can also set each corner separately using:

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius
div {
  border: 2px solid #333;
  border-top-left-radius: 20px;
  border-top-right-radius: 5px;
  border-bottom-right-radius: 20px;
  border-bottom-left-radius: 5px;
}

Circle and Oval Shapes

Setting border-radius: 50% makes elements appear as circles or ovals (depending on width and height).

div {
  width: 100px;
  height: 100px;
  background-color: teal;
  border-radius: 50%;
}
Tip: Use border-radius creatively to build cards, buttons, profile images, and even chat bubbles!

CSS Spacing

CSS spacing refers to the space inside and outside elements. You can control spacing using two main properties:

  • margin – controls space outside the element
  • padding – controls space inside the element

1) margin

The margin property adds space around an element, outside of its border.

Example:

div {
  margin: 20px;
}

This applies 20px of space on all four sides.

Individual Margins

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
div {
  margin-top: 10px;
  margin-right: 15px;
  margin-bottom: 20px;
  margin-left: 25px;
}

2) padding

The padding property adds space between the content and the border of an element.

Example:

div {
  padding: 20px;
}

This adds 20px space inside the element on all sides.

Individual Padding

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
div {
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 20px;
  padding-left: 25px;
}

3) Shorthand Syntax

You can use one line to define spacing for all four sides:

/* top | right | bottom | left */
margin: 10px 20px 30px 40px;
padding: 5px 10px 5px 10px;

4) Auto Margin (Centering)

Use margin: auto; to horizontally center a block element (like a div):

div {
  width: 300px;
  margin: auto;
}
Tip: Padding affects the inside space of an element. Margin affects the outside space between elements.

CSS Display

The display property in CSS determines how an element is displayed on the page. It's one of the most important layout-related properties.

Common display Values

  • block – The element takes up the full width available.
  • inline – The element does not start on a new line and only takes up as much width as necessary.
  • inline-block – Like inline, but allows setting width and height.
  • none – Hides the element completely (it won’t be visible or take up space).
  • flex – Makes the container a flexbox for layout purposes.
  • grid – Makes the container a grid layout.

1) display: block;

Block elements start on a new line and stretch to full width.

div {
  display: block;
}

2) display: inline;

Inline elements do not start on a new line and only occupy as much width as needed.

span {
  display: inline;
}

3) display: inline-block;

Inline-block elements behave like inline elements but allow setting width and height.

button {
  display: inline-block;
  width: 100px;
  height: 40px;
}

4) display: none;

This completely hides the element and removes it from the layout flow.

p.hidden {
  display: none;
}

5) display: flex;

Turns a container into a flexbox for aligning and distributing items in rows or columns.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

6) display: grid;

Turns a container into a grid layout. Useful for creating complex designs.

.grid-container {
  display: grid;
  grid-template-columns: auto auto;
  gap: 10px;
}
Note: Use display: none; to hide elements. Use visibility: hidden; to hide an element but keep its space.

CSS Buttons

You can style HTML buttons using CSS to improve appearance and interactivity. You can modify background, color, padding, borders, hover effects, and more.

Basic Button Styling

Example:

button {
  background-color: #4CAF50;
  color: white;
  padding: 12px 24px;
  border: none;
  border-radius: 6px;
  cursor: pointer;
}

Hover Effect

Add hover styles for interaction feedback:

button:hover {
  background-color: #45a049;
}

Outlined Button

Example:

button {
  background-color: white;
  color: #4CAF50;
  border: 2px solid #4CAF50;
  padding: 10px 20px;
  border-radius: 6px;
}

Disabled Button

Style a disabled button using the :disabled pseudo-class:

button:disabled {
  background-color: #ccc;
  color: #666;
  cursor: not-allowed;
}

Rounded & Large Button

Example:

button {
  padding: 16px 32px;
  border-radius: 50px;
  font-size: 18px;
}
Tip: You can also use transitions, gradients, and shadows to make buttons more interactive and attractive.

CSS Float

The float property in CSS is used to position elements to the left or right, allowing text or inline elements to wrap around them.

Float Values

  • left – Floats the element to the left
  • right – Floats the element to the right
  • none – Default, element does not float

Example: Image Floating

img {
  float: right;
  margin-left: 20px;
}

You can float an image to the right and let text wrap around it.

Clear Property

Use the clear property to stop an element from wrapping around floated elements.

.clearfix {
  clear: both;
}

CSS Font

The font properties allow you to style the appearance of text, including font family, size, weight, style, and more.

Common Font Properties

  • font-family – Sets the font type
  • font-size – Sets the size of the font
  • font-weight – Sets how bold the font is
  • font-style – Sets italic, normal, or oblique
  • font-variant – Small-caps, normal

Example

p {
  font-family: 'Verdana', sans-serif;
  font-size: 18px;
  font-weight: bold;
  font-style: italic;
}

Font Shorthand

You can use shorthand to combine all font properties into a single line:

p {
  font: italic small-caps bold 16px/1.5 'Georgia', serif;
}
Tip: Always include a fallback font in font-family (e.g., Arial, sans-serif) in case the first font is unavailable.

CSS Colors

CSS allows you to set colors for text, backgrounds, borders, and more using the color and background-color properties.

Color Types

  • Named Colors: red, blue, green, etc.
  • Hex Values: #ff0000, #00ff00
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5)
  • HSL: hsl(0, 100%, 50%)

Example

body {
  background-color: lightblue;
}

h1 {
  color: navy;
}

This is a heading

This is a sample background and text color.

Tip: Use rgba() to add transparency to color.

CSS Margin

The margin property creates space around elements, outside of any borders. It’s used to separate elements visually.

Set All Margins

div {
  margin: 20px;
}

This sets 20px space on all four sides.

Individual Margins

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
div {
  margin-top: 10px;
  margin-right: 15px;
  margin-bottom: 20px;
  margin-left: 25px;
}

Shorthand Margin

Use one line to set all four margins:

margin: 10px 20px 30px 40px;
/* top right bottom left */

Auto Margin (Centering)

div {
  width: 300px;
  margin: auto;
}
Tip: margin: auto; is often used to center block elements horizontally.

CSS Overflow

The overflow property controls what happens to content that overflows an element's box.

Values

  • visible – Default. Content is not clipped and may overflow.
  • hidden – Overflow is clipped; the rest is hidden.
  • scroll – Always adds scrollbars.
  • auto – Adds scrollbars only if necessary.

Example

div {
  width: 200px;
  height: 100px;
  overflow: auto;
}
Tip: Use overflow-x and overflow-y for horizontal and vertical control.

CSS Padding

The padding property creates space inside the element, between the content and the border.

Example

div {
  padding: 20px;
}

Individual Padding

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Shorthand Example

padding: 10px 20px 30px 40px;
/* top right bottom left */

CSS Position

The position property specifies how an element is positioned in a document.

Position Types

  • static – Default positioning (normal flow)
  • relative – Positioned relative to its normal position
  • absolute – Positioned relative to the nearest positioned ancestor
  • fixed – Positioned relative to the viewport
  • sticky – Switches between relative and fixed based on scroll

Example

div {
  position: absolute;
  top: 50px;
  left: 100px;
}

CSS Outline

The outline property draws a line outside the element's border to highlight it without affecting layout.

Basic Outline

div {
  outline: 2px solid red;
}

Outline Offset

Add space between the outline and border:

div {
  outline-offset: 5px;
}

CSS Visibility

The visibility property specifies whether an element is visible or hidden, but it still takes up space in the layout.

Values

  • visible – Default (element is shown)
  • hidden – Element is hidden, but space is preserved
  • collapse – Used with table rows/columns

Example

p.hidden {
  visibility: hidden;
}
Note: Unlike display: none, visibility: hidden keeps the element's space.

CSS Counters

CSS counters are variables maintained by CSS to automatically number elements, like sections or list items.

How It Works

  1. Initialize the counter using counter-reset.
  2. Increment the counter using counter-increment.
  3. Display the counter using content: counter(name) in ::before or ::after.

Example

body {
  counter-reset: section;
}

h2::before {
  counter-increment: section;
  content: "Section " counter(section) ": ";
}

This will number all <h2> headings as Section 1, Section 2, etc.

Tip: Counters work best with pseudo-elements like ::before.

CSS Icons

You can add icons to your web page using CSS libraries like Font Awesome, or by using Unicode or SVG images.

Using Font Awesome

First, include the Font Awesome CDN in your HTML:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">

Then use the icon class in an element:

<i class="fa fa-home"></i> Home

Using Unicode

p::before {
  content: "\2713"; /* Checkmark */
  color: green;
  margin-right: 5px;
}

Using SVG

You can also use inline SVGs for scalable, styleable icons.

<svg width="24" height="24">
  <circle cx="12" cy="12" r="10" stroke="black" stroke-width="2" fill="blue" />
</svg>
Tip: Font Awesome provides thousands of free icons with easy integration.
